# Habit Tracker

Минималистичный чёрно‑белый трекер привычек: бинарные привычки, колесо жизни,
утренние/вечерние чек‑апы, месячная тепловая карта, числовой «psycho mode» и
лёгкая геймификация (опыт/уровни → мини‑игры) с онбордингом.

**Без зависимостей и сборки** — vanilla HTML/CSS/JS, данные в `localStorage`.

🔗 **Демо:** https://iamgintonyc.github.io/habit-tracker/

## Запуск локально

```
python3 -m http.server 4599
```

Открыть http://localhost:4599
